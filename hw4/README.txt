#### TO RUN ####
Extract contents into dir on flip.
Inside the directory on flip,

type: virtualenv hw4 (wait for completion)
type: source hw4/bin/activate (should see (hw4) on left side once VENV is working)
type: pip install -r requirements.txt (I have included the requirements.txt file in my zip)


Generate QR: python ./lloyddo-MP4.py --generate-qr
Get OTP: python ./lloyddo-MP4.py --get-otp

To leave venv: deactivate
